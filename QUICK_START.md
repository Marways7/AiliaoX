# 🚀 AiliaoX 快速启动指南

## ✨ 一键启动（推荐）

只需一个命令，自动完成所有启动流程：

```bash
npm start
```

**启动脚本会自动完成：**
- ✅ 检查运行环境（Node.js版本）
- ✅ 启动数据库服务（MySQL + Redis）
- ✅ 运行数据库迁移
- ✅ 生成Prisma Client
- ✅ 启动前后端开发服务器
- ✅ 显示炫酷的启动动画和进度条

## 🎯 访问地址

启动成功后，您可以访问：

- **前端界面**: http://localhost:5173
- **后端API**: http://localhost:3000
- **健康检查**: http://localhost:3000/health
- **Prisma Studio**: 运行 `npm run prisma:studio`

## 🛠️ 其他命令

### 开发相关
```bash
# 仅启动前后端（不启动数据库）
npm run dev

# 仅启动后端
npm run dev:backend

# 仅启动前端
npm run dev:frontend
```

### 数据库管理
```bash
# 启动数据库容器
npm run docker:up

# 停止数据库容器
npm run docker:down

# 查看数据库日志
npm run docker:logs

# 运行数据库迁移
npm run prisma:migrate

# 打开Prisma Studio（数据库可视化工具）
npm run prisma:studio
```

### 构建与测试
```bash
# 构建生产版本
npm run build

# 运行测试
npm run test

# 代码检查
npm run lint
```

## 🔧 环境要求

- **Node.js**: >= 20.0.0
- **npm**: >= 10.0.0
- **Docker**: 可选（推荐）

## 🎨 功能特点

- 🎭 **炫酷启动动画**: ASCII艺术横幅 + 彩色输出
- ⚡ **进度条显示**: 实时显示每个步骤的进度
- 🔍 **智能检查**: 自动检查环境配置
- 🚀 **一键启动**: 无需手动执行多个命令
- 📊 **清晰反馈**: 每个步骤都有详细的状态提示

## 💡 提示

- 首次启动可能需要等待几秒钟，让服务完全启动
- 如有问题，请检查终端输出的日志信息
- 按 `Ctrl+C` 可以优雅地停止所有服务
